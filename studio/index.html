how stacks work
  • drop a file on a stack to add inside it; drop on empty page for a new song
  • drag one song onto another’s center to combine into a stack
  • drag the move handle to reorder blocks; drag track rows inside a stack to reorder tracks
  • × on a track removes it; delete removes the whole block
  • stack cover and name are set on the stack in the page (drop or choose image)
